# BathLARP Storybook

## Themes

This is where all the basic styling information is stored; colours, typography and anything else that
makes more sense to be configured at a root level.

## Display

This is for all components that are purely for display purposes, such as icons and event calendar cards.

## Interactive

This is for all interactive components, such as buttons and form inputs.

## Layout

This is specifically for page layout elements, such as headers, footers and menus.
